# Responsive Components Example

This sample demonstrates a responsive mobile card for fleet-style summary data.

It is public-safe and uses fake labels only.

## Demonstrates

- Responsive layout based on window size.
- Compact card design.
- Safe placeholder status values.
- Reusable props for dashboard-style summaries.

## Redaction Notes

- Do not include real vehicle identifiers.
- Do not include real driver or customer data.
- Do not include real business regions or operational status values.
