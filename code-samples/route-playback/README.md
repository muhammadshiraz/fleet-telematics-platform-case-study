# Route Playback Controls Example

This sample demonstrates a route playback control pattern with fake route points.

It is intentionally simplified for public sharing and does not copy production tracking code.

## Demonstrates

- Play, pause, reset controls.
- Current route-point index.
- Synthetic speed and timestamp labels.
- Safe placeholder route data.

## Redaction Notes

- Do not include real vehicle routes.
- Do not include real timestamps tied to real movement.
- Do not include GPS points from production.
- Do not include driver names, vehicle IDs, or registration numbers.
