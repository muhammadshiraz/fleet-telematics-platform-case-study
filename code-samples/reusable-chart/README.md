# Reusable Chart Example

This sample demonstrates a simplified public-safe bar chart component for React Native.

It is inspired by the app's reusable chart pattern, but it is not copied from production code. The data, labels, styling, and component names are synthetic.

## Demonstrates

- Reusable chart props.
- Placeholder fleet/safety metrics.
- Empty-state handling.
- Responsive width handling.

## Redaction Notes

- Do not use production report labels.
- Do not include real violation counts.
- Do not include real driver, vehicle, region, or customer data.
