# Animated Map Marker Example

This sample demonstrates a public-safe animated vehicle marker using fake coordinates.

It is a simplified example and does not copy production tracking code.

## Demonstrates

- `AnimatedRegion` for smooth marker movement.
- Marker rotation from a synthetic heading value.
- Fake vehicle status labels.
- Map rendering with demo-only coordinates.

## Redaction Notes

- Do not use real GPS coordinates.
- Do not use real route history.
- Do not use real vehicle identifiers or registration numbers.
- Do not publish production marker icons if they are private brand assets.
