# Redux Async Flow Example

This sample demonstrates a public-safe Redux Thunk request, success, and failure flow.

It is rewritten from scratch and does not include production endpoints, headers, payloads, or response data.

## Demonstrates

- Action types.
- Thunk action creator.
- Axios call to a placeholder URL.
- Reducer loading, data, and error states.

## Redaction Notes

- Use `https://api.example.invalid` or a mock client in public docs.
- Never include production API hosts.
- Never include auth headers, tokens, or private payload shapes.
